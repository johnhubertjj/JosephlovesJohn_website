# SEO Admin Notes

## Indexing intent

- `index,follow`
  - `/`
  - `/intro/`
  - `/music/`
  - `/art/`
  - `/contact/`
  - legal pages
  - `/mastering-services/`

- `noindex,follow`
  - shop login
  - register
  - password reset
  - checkout
  - account
  - order success pages

## When updating content

- Keep page titles specific to the route.
- Keep meta descriptions short and human-readable.
- Prefer real alt text for artwork/gig photos even though fallbacks exist.
- If a new public landing page is added, include it in `sitemap.xml` and give it SEO metadata.

## After major site changes

- Re-check `robots.txt`
- Re-check `sitemap.xml`
- Use Search Console URL Inspection on the changed page
